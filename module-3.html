1 Introduction to State Space Concepts

THEORY Dec 2019Sep 2020

The methods of analysis and design of feedback control systems such as root locus, Bode, and Nyquist plots require the physical system to be modeled in the form of a transfer function. Though the transfer function approach is simple, it suffers from several severe drawbacks.

Drawbacks of Transfer Functions

  • Transfer function models are only applicable to linear time-invariant (LTI) systems.
  • A transfer function is only defined under zero initial conditions.
  • It reveals only the output of the system for a given input, providing no information regarding the internal states of the system.
  • It is restricted to Single-Input Single-Output (SISO) systems (for basic analysis).

Advantages of State Space Analysis

  • It is a direct time-domain approach, making it highly suitable for digital computer computations.
  • It is a powerful technique for the design and analysis of linear, nonlinear, time-variant, time-invariant, SISO, and MIMO systems.
  • An $n^{th}$ order differential equation is expressed as $n$ first-order differential equations, making mathematical solutions easier.
  • Initial conditions can be easily incorporated into the equations.
  • Allows for the optimal design of modern controllers.

Fundamental Definitions

1. State:

The state of a dynamic system is the smallest set of variables (called state variables) such that knowledge of these variables at $t = t_0$, together with knowledge of the input for $t \ge t_0$, completely determines the behavior of the system for any time $t \ge t_0$.

2. State Variables:

The variables making up the smallest set of variables that determine the state of the dynamic system. They need not be physically measurable or observable quantities (e.g., intermediate mathematical derivatives can be state variables).

3. State Vector:

If $n$ state variables are needed to completely describe the behavior of a given system, these $n$ variables can be considered the $n$ components of a vector $X(t)$. This vector uniquely determines the system state.

4. State Space:

The $n$-dimensional space whose coordinate axes consist of the $x_1$ axis, $x_2$ axis, $\dots$, $x_n$ axis is called the state space. Any state can be represented by a point in this state space.

2 State Space Representation & Block Diagram

In state space analysis, we are concerned with three types of variables: input variables ($u$), output variables ($y$), and state variables ($x$). The state-space representation is not unique; a system can have infinite valid state-space representations, but the number of state variables ($n$) is always the same.

Linear Time-Invariant (LTI) Systems

If the system equations are linearized about an operating point and do not involve time $t$ explicitly, the system is LTI. The state and output equations are:

State Equation

$$ \dot{X}(t) = A X(t) + B U(t) $$

Output Equation

$$ Y(t) = C X(t) + D U(t) $$
  • $X(t)$: State Vector ($n \times 1$)
  • $U(t)$: Input Vector ($m \times 1$)
  • $Y(t)$: Output Vector ($p \times 1$)
  • $A$: System / State Matrix ($n \times n$)
  • $B$: Input Matrix ($n \times m$)
  • $C$: Output Matrix ($p \times n$)
  • $D$: Feedforward Matrix ($p \times m$)

Physical System Modeling: RLC Circuit

Question: Derive the state space model of the RLC circuit shown below. The input voltage is $u(t)$ and the output is $y(t)$. Select the inductor currents $i_1(t)$ and $i_2(t)$ as state variables.

Step 1: Identify States and Apply Kirchhoff's Laws

Let state variables be $x_1 = i_1$ and $x_2 = i_2$.

Applying KVL in the first (left) loop:

$$ u(t) = L_1 \frac{di_1}{dt} + R_1 i_1 + L_2 \frac{di_2}{dt} + R_2 i_2 $$

Substitute values ($L_1=1, L_2=1, R_1=1, R_2=1$):

$$ u(t) = \frac{di_1}{dt} + i_1 + \frac{di_2}{dt} + i_2 \quad \dots \text{(Eq. 1)} $$

Applying KVL in the second (right) loop (or analyzing node currents):

The current through the output resistor is $i_{out} = i_1 - i_2$. The voltage across the middle branch equals the voltage across the right branch:

$$ L_2 \frac{di_2}{dt} + R_2 i_2 = R_3 (i_1 - i_2) $$

Substitute values ($L_2=1, R_2=1, R_3=1$):

$$ \frac{di_2}{dt} + i_2 = 1(i_1 - i_2) \implies \frac{di_2}{dt} = i_1 - 2i_2 \quad \dots \text{(Eq. 2)} $$

Step 2: Formulate State Equations

From Eq. 2, we directly get the state equation for $x_2$:

$$ \dot{x}_2 = x_1 - 2x_2 $$

Substitute $\frac{di_2}{dt}$ from Eq. 2 into Eq. 1 to find $\dot{x}_1$:

$$ u(t) = \frac{di_1}{dt} + i_1 + (i_1 - 2i_2) + i_2 $$ $$ u(t) = \frac{di_1}{dt} + 2i_1 - i_2 $$

$$ \dot{x}_1 = -2x_1 + x_2 + u $$

Step 3: Final Matrix Representation

The output is the voltage across $R_3$:

$$ y(t) = R_3(i_1 - i_2) = 1(x_1 - x_2) = x_1 - x_2 $$

Matrix form:

$$ \begin{bmatrix} \dot{x}_1 \\ \dot{x}_2 \end{bmatrix} = \begin{bmatrix} -2 & 1 \\ 1 & -2 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} + \begin{bmatrix} 1 \\ 0 \end{bmatrix} u $$
$$ y = \begin{bmatrix} 1 & -1 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} $$

Correlation: State Space to Transfer Function

Taking the Laplace transform of the state equations assuming zero initial conditions ($X(0)=0$):

$$ s X(s) = A X(s) + B U(s) $$ $$ (sI - A) X(s) = B U(s) \implies X(s) = (sI - A)^{-1} B U(s) $$

Substitute $X(s)$ into the output equation $Y(s) = C X(s) + D U(s)$:

$$ G(s) = \frac{Y(s)}{U(s)} = C (sI - A)^{-1} B + D $$

Note: The denominator of the transfer function is given by the determinant $|sI - A|$. Setting $|sI - A| = 0$ yields the characteristic equation of the system. The roots of this equation are the poles of the transfer function, which are exactly the eigenvalues of the system matrix $A$.

3 Canonical Forms of State Representation

HIGHLY EXAMINED Dec 2019Sep 2020Jul 2021

The state variable representation is not unique. For a given transfer function, we can define infinite sets of state variables. However, certain standard forms (canonical forms) are highly useful for controller and observer design, or for decoupling states.

3.1 Controllable Canonical Form (Phase Variable Form)

This form is structured such that the coefficients of the transfer function's denominator appear directly in the last row of the $A$ matrix, and the numerator coefficients appear in the $C$ matrix. It guarantees the system is state controllable.

For a general transfer function:

$$ \frac{Y(s)}{U(s)} = \frac{b_0 s^n + b_1 s^{n-1} + \dots + b_n}{s^n + a_1 s^{n-1} + \dots + a_n} $$

The state matrices are written by inspection:

$$ A = \begin{bmatrix} 0 & 1 & 0 & \dots & 0 \\ 0 & 0 & 1 & \dots & 0 \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ -a_n & -a_{n-1} & -a_{n-2} & \dots & -a_1 \end{bmatrix}, \quad B = \begin{bmatrix} 0 \\ 0 \\ \vdots \\ 1 \end{bmatrix} $$
$$ C = \begin{bmatrix} (b_n - a_n b_0) & (b_{n-1} - a_{n-1} b_0) & \dots & (b_1 - a_1 b_0) \end{bmatrix} $$

3.2 Observable Canonical Form

This form is the mathematical dual of the controllable canonical form. The $A$ matrix is the transpose of the controllable $A$ matrix. The coefficients of the denominator appear in the last column. It guarantees the system is completely observable.

$$ A = \begin{bmatrix} 0 & 0 & \dots & -a_n \\ 1 & 0 & \dots & -a_{n-1} \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & -a_1 \end{bmatrix}, \quad B = \begin{bmatrix} b_n - a_n b_0 \\ b_{n-1} - a_{n-1} b_0 \\ \vdots \\ b_1 - a_1 b_0 \end{bmatrix} $$
$$ C = \begin{bmatrix} 0 & 0 & \dots & 1 \end{bmatrix} $$

3.3 Diagonal Canonical Form (Modal Form)

Used when the system has real and distinct poles. The transfer function is expanded using partial fractions. The resulting state equations are entirely decoupled (each state derivative depends only on itself and the input).

$$ G(s) = \frac{Y(s)}{U(s)} = \frac{C_1}{s + p_1} + \frac{C_2}{s + p_2} + \dots + \frac{C_n}{s + p_n} $$
$$ A = \begin{bmatrix} -p_1 & 0 & \dots & 0 \\ 0 & -p_2 & \dots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & -p_n \end{bmatrix}, \quad B = \begin{bmatrix} 1 \\ 1 \\ \vdots \\ 1 \end{bmatrix}, \quad C = \begin{bmatrix} C_1 & C_2 & \dots & C_n \end{bmatrix} $$

Block Diagram representation showing parallel, decoupled state paths.

3.4 Jordan Canonical Form

Used when the system has repeated roots. A full diagonal matrix is impossible, so a Jordan block is formed with the repeated root on the main diagonal and 1s on the super-diagonal.

For a transfer function with a repeated pole $p_1$ of multiplicity 3:

$$ G(s) = \frac{C_1}{(s + p_1)^3} + \frac{C_2}{(s + p_1)^2} + \frac{C_3}{s + p_1} + \dots $$
$$ A = \begin{bmatrix} -p_1 & 1 & 0 \\ 0 & -p_1 & 1 \\ 0 & 0 & -p_1 \end{bmatrix}, \quad B = \begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix}, \quad C = \begin{bmatrix} C_1 & C_2 & C_3 \end{bmatrix} $$

4 Solution of State Equations & State Transition Matrix

CORE MATHEMATICS Apr 2018May 2019Sep 2020

Solving the state equation $\dot{X}(t) = A X(t) + B U(t)$ provides the time response of the internal states of the system.

4.1 Time Domain Solution

The complete solution consists of two parts: the Zero-Input Response (due to initial conditions) and the Zero-State Response (due to the input).

$$ X(t) = \underbrace{e^{At} X(0)}_{\text{Zero-Input Response}} + \underbrace{\int_{0}^{t} e^{A(t-\tau)} B U(\tau) d\tau}_{\text{Zero-State (Forced) Response}} $$

The matrix exponential $e^{At}$ is known as the State Transition Matrix (STM), denoted by $\Phi(t)$.

4.2 Properties of State Transition Matrix $\Phi(t)$

  • 1. Identity at t=0: $\Phi(0) = e^{A \cdot 0} = I$
  • 2. Inverse Property: $\Phi(-t) = \Phi^{-1}(t)$
  • 3. Multiplicative Property: $\Phi(t_1 + t_2) = \Phi(t_1)\Phi(t_2)$
  • 4. Power Property: $[\Phi(t)]^n = \Phi(nt)$
  • 5. State Transition Property: $\Phi(t_2 - t_1)\Phi(t_1 - t_0) = \Phi(t_2 - t_0)$

4.3 Solution using Laplace Transform

Taking the Laplace transform of $\dot{X}(t) = A X(t) + B U(t)$:

$$ s X(s) - X(0) = A X(s) + B U(s) $$ $$ (sI - A) X(s) = X(0) + B U(s) $$ $$ X(s) = (sI - A)^{-1} X(0) + (sI - A)^{-1} B U(s) $$

Taking the inverse Laplace transform gives the time-domain solution:

$$ X(t) = L^{-1}\{(sI - A)^{-1}\} X(0) + L^{-1}\{(sI - A)^{-1} B U(s)\} $$

From this, it is evident that the State Transition Matrix can be computed using Laplace transforms:

$$ \Phi(t) = e^{At} = L^{-1}\{(sI - A)^{-1}\} $$

Problem: Complete Response of a System

Question: Find the complete response of the system:
$\dot{X} = \begin{bmatrix} 0 & 1 \\ -2 & -3 \end{bmatrix} X + \begin{bmatrix} 2 & 1 \\ 0 & 1 \end{bmatrix} U(t), \quad X(0) = \begin{bmatrix} 0 \\ 0 \end{bmatrix}$
$Y(t) = \begin{bmatrix} 1 & 0 \\ 1 & 1 \end{bmatrix} X$, for the input $U(t) = \begin{bmatrix} u(t) \\ e^{3t}u(t) \end{bmatrix}$, where $u(t)$ is the unit step input.

Step 1: Identify Matrices and Resolve $(sI-A)^{-1}$

$$ A = \begin{bmatrix} 0 & 1 \\ -2 & -3 \end{bmatrix}, B = \begin{bmatrix} 2 & 1 \\ 0 & 1 \end{bmatrix}, C = \begin{bmatrix} 1 & 0 \\ 1 & 1 \end{bmatrix}, U(s) = \begin{bmatrix} 1/s \\ \frac{1}{s-3} \end{bmatrix} $$

Find $(sI - A)$:

$$ sI - A = \begin{bmatrix} s & 0 \\ 0 & s \end{bmatrix} - \begin{bmatrix} 0 & 1 \\ -2 & -3 \end{bmatrix} = \begin{bmatrix} s & -1 \\ 2 & s+3 \end{bmatrix} $$

Find the inverse $(sI - A)^{-1} = \frac{\text{Adj}(sI-A)}{|sI-A|}$:

$$ |sI - A| = s(s+3) - (-2) = s^2 + 3s + 2 = (s+1)(s+2) $$
$$ (sI - A)^{-1} = \frac{1}{(s+1)(s+2)} \begin{bmatrix} s+3 & 1 \\ -2 & s \end{bmatrix} = \begin{bmatrix} \frac{s+3}{(s+1)(s+2)} & \frac{1}{(s+1)(s+2)} \\ \frac{-2}{(s+1)(s+2)} & \frac{s}{(s+1)(s+2)} \end{bmatrix} $$

Step 2: Calculate State Vector $X(s)$

Since $X(0) = 0$, the zero-input response is 0. The state vector is purely the zero-state response:

$$ X(s) = (sI - A)^{-1} B U(s) $$
$$ X(s) = \begin{bmatrix} \frac{s+3}{(s+1)(s+2)} & \frac{1}{(s+1)(s+2)} \\ \frac{-2}{(s+1)(s+2)} & \frac{s}{(s+1)(s+2)} \end{bmatrix} \begin{bmatrix} 2 & 1 \\ 0 & 1 \end{bmatrix} \begin{bmatrix} 1/s \\ \frac{1}{s-3} \end{bmatrix} $$

First, multiply $(sI-A)^{-1}$ and $B$:

$$ = \begin{bmatrix} \frac{2(s+3)}{(s+1)(s+2)} & \frac{s+4}{(s+1)(s+2)} \\ \frac{-4}{(s+1)(s+2)} & \frac{s-2}{(s+1)(s+2)} \end{bmatrix} \begin{bmatrix} 1/s \\ \frac{1}{s-3} \end{bmatrix} $$

Now multiply by $U(s)$:

$$ X(s) = \begin{bmatrix} \frac{2s+6}{s(s+1)(s+2)} + \frac{s+4}{(s+1)(s+2)(s-3)} \\ \frac{-4}{s(s+1)(s+2)} + \frac{s-2}{(s+1)(s+2)(s-3)} \end{bmatrix} = \begin{bmatrix} X_1(s) \\ X_2(s) \end{bmatrix} $$

Step 3: Partial Fraction Expansion & Inverse Laplace

Applying partial fractions to each term in $X(s)$:

$$ X_1(s) = \left( \frac{3}{s} - \frac{4}{s+1} + \frac{1}{s+2} \right) + \left( \frac{-3/4}{s+1} + \frac{1/5}{s+2} + \frac{7/20}{s-3} \right) $$
$$ X_2(s) = \left( \frac{-2}{s} + \frac{4}{s+1} - \frac{2}{s+2} \right) + \left( \frac{3/4}{s+1} - \frac{2/5}{s+2} + \frac{1/20}{s-3} \right) $$

Taking $L^{-1}$ yields $X(t)$:

$$ X(t) = \begin{bmatrix} 3u(t) - \frac{19}{4}e^{-t} + \frac{6}{5}e^{-2t} + \frac{7}{20}e^{3t} \\ -2u(t) + \frac{19}{4}e^{-t} - \frac{12}{5}e^{-2t} + \frac{1}{20}e^{3t} \end{bmatrix} $$

Step 4: Calculate Output $Y(t)$

$Y(t) = C X(t)$

$$ Y(t) = \begin{bmatrix} 1 & 0 \\ 1 & 1 \end{bmatrix} \begin{bmatrix} X_1(t) \\ X_2(t) \end{bmatrix} = \begin{bmatrix} X_1(t) \\ X_1(t) + X_2(t) \end{bmatrix} $$

Final Output Matrix:

$$ Y(t) = \begin{bmatrix} 3u(t) - \frac{19}{4}e^{-t} + \frac{6}{5}e^{-2t} + \frac{7}{20}e^{3t} \\ u(t) - \frac{6}{5}e^{-2t} + \frac{2}{5}e^{3t} \end{bmatrix} $$
